## 1. Термины и определения

1.1. АПК - аппаратно-программный комплекс.

1.2. АСУНО - автоматизированная система управления наружным освещением.

1.3. ШУНО - шкаф управления наружным освещением.

1.4. IoT - Internet of Things, Интернет вещей — это концепция, при которой физические устройства подключены к интернету, собирают данные, обмениваются ими и могут автоматически выполнять действия без постоянного участия человека.

1.5. LoRaWAN - Long Range Wide Area Network, протокол беспроводной связи для построения сетей Интернета вещей (IoT). В рамках АПК используется для организации связи со светильниками по радиоканалу.

1.6. PLC - Power Line Communication, технология для передачи цифровой информации по проводам, которые одновременно используются для передачи электроэнергии.

1.7. DALI, DALI-2 - цифровые протоколы для организации проводной системы управления освещением.

1.8. D4i - стандарт, расширяющий DALI-2 IoT‑функциями и дополнительными возможностями диагностики светильников.

1.9. LLM - Large Language Model, большая языковая модель, используемая в АПК в качестве искусственного интеллекта (ИИ).

## 2. Назначение

2.1. АПК предназначен для создания автоматизированной системы управления наружным освещением (АСУНО).

### 4.2. LoRaWAN-модули

4.2.1. LoRaWAN-модули - устройства индивидуального управления светильниками, устанавливаемые непосредственно на светильниках и обеспечивающие беспроводную связь с остальным АПК по технологии LoRaWAN.

4.2.2. Модули устанавливаются на светильники посредством Zhaga-разъема.

4.2.3. Модули обеспечивают автономную автоматическую работу:

- по расписанию
- по восходу/закату
- по датчику освещенности

4.2.4. Выбор режима и его настройка осуществляется пользователями через клиенты АПК.

4.2.5. Модули поддерживают индивидуальное и групповое оперативное управление яркостью светильников.

4.2.6. Оперативное управление осуществляется с помощью клиентов АПК.

### 4.3. PLC-модули

4.3.1. Модули устанавливаются в светильники или в непосредственной близости от них.

4.3.2. Модули соединяются со светильниками силовыми линиями 220В и кабелями управления.

### 4.4. ШУНО

4.4.1. Шкафы управления наружным освещением (ШУНО) - комплекты оборудования АСУНО, поддерживающие связь, с одной стороны, со светильниками по LoRaWAN, с другой - с сервером АПК через GSM/TCP/IP-соединение. В состав ШУНО входят:
4.4.2. ШУНО - металлический шкаф в пылевлагозащищенном антивандальном исполнении.

4.4.3. Дверь шкафа оборудуется датчиком открытия.

4.4.4. Оборудование шкафа:

4.4.4.1. **Контроллер AWADA RAPIDA** - управляет светильниками, подключенными к шкафу; взаимодействует с сервером АПК.

4.4.4.2. **Выключатели** - стоят на силовых линиях, питающих светильники; имеют функционал телемеханики (состояние, включение/выключение).

4.4.4.3. **Счетчик электроэнергии** - телеметрия энергопотребления системы освещения.

4.4.4.4. **GSM модем** - TCP/IP-связь контроллера с сервером АПК.

4.4.4.5. **База LoRaWAN** - беспроводная связь контроллера с модулями LoRaWAN светильников.

4.4.4.6. **База PLC** - связь контроллера с модулями PLC светильников по линиям 220В.

### 4.5. Сервер АПК

4.5.1. Интеграция со сторонними сервисами.

4.5.2. Является веб-сервером для клиентов АПК.

4.5.3. Взаимодействует с контроллерами RAPIDA ШУНО.

4.5.4. Хранит базу данных АПК: текущую конфигурацию АСУНО, настройки и текущее состояние устройств, архив.

4.5.5. Является сервером системы ограничения прав доступа.

4.5.6. Является сервером LLM.

### 4.6. Клиенты АПК

4.6.1. Нативные: ПК, Android/iOS.
4.6.2. Интеграция с текущим клиентом.

4.6.3. Клиент - веб-приложение, работающее в браузере на устройстве пользователя.

4.6.4. В качестве веб-сервера использует сервер АПК.

4.6.5. Имеет интерфейс, адаптированный под использование на ПК, планшете, телефоне.

4.6.6. Обеспечивает:

4.6.6.1. Доступ к данным согласно правам доступа пользователя.

4.6.6.2. Настройку и оперативное управление оборудованием.

4.6.6.3. Изменение конфигурации АСУНО (добавление/удаление/изменение светильников/ШУНО/линий электропередачи и т.п.).

4.6.6.4. Формирование отчетности по работе АСУНО.
