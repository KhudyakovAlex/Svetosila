# Автоматизация проекта Светосила

## Структура файлов

```
PROJECT/
├── log.md                          # Дневник коротышек
├── status.md                       # Статусы задач
├── update_dates.py                 # Автообновление дат в MD
└── INDEX/
    ├── index.html                  # Главная страница (генерируется)
    └── update_from_project.py      # Синхронизация PROJECT → index.html
```

## Редактирование

### Дневник коротышек (`log.md`)

Формат записей:
```
YYYY-MM-DD HH:MM — текст записи
```

Пример:
```
2025-12-18 12:00 — Надо уже окончательно определяться с тем, чью карту будем юзать.
```

### Статусы (`status.md`)

Формат:
```markdown
# Текущий статус

## Отложено на <s>завтра</s> послезавтра

- Задача 1
- Задача 2

## Прям сейчас делаем

- Задача 3
- Задача 4

## Готовченко

- Готовая задача 1
- Готовая задача 2
```

## Автоматическое обновление

### При коммите (Git hook)

При выполнении `git commit` автоматически:
1. Обновляются даты в MD-файлах (PDS/, PRD/)
2. `log.md` и `status.md` синхронизируются с `index.html`
3. Обновленный `index.html` добавляется в коммит

### Ручной запуск

Обновить `index.html` из `PROJECT/*.md`:
```bash
python PROJECT\INDEX\update_from_project.py
```

Обновить даты в MD-файлах:
```bash
python PROJECT\update_dates.py
```

## Git Hook

Hook установлен в `.git/hooks/pre-commit` и запускается автоматически перед каждым коммитом.

Если нужно пропустить hook (не рекомендуется):
```bash
git commit --no-verify
```
